# Car Meetup Events Website

This project is a static website developed as part of my Web Technology coursework.

## 📌 Description
The website is designed to showcase car meetup events, allowing users to explore upcoming events, galleries, videos, and community features in a visually engaging way.

## 🎯 Objective
- Apply HTML & CSS concepts in a real-world scenario  
- Design an event-based website interface  
- Understand layout structuring and responsiveness  

## 🚀 Features
- Responsive homepage with event highlights  
- Upcoming car meetup events section  
- Image gallery showcasing event moments  
- Embedded event videos  
- Membership join form (UI-level)  
- Layout built using Flexbox & Grid  

## 🌱 Learning Outcomes
- Semantic HTML structuring  
- CSS layout techniques (Flexbox & Grid)  
- Media integration (images & videos)  
- Designing user-friendly static interfaces  

## 🛠️ Technologies Used
- HTML5  
- CSS3
