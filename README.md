# 🚗 CarMeet — Car Meetup Events Website

> A visually engaging static website to discover, explore, and celebrate car meetup culture.

---

## 📖 About

**CarMeet** is a front-end web project built as part of the Web Technology coursework at **Savitribai Phule Pune University**. The site brings the excitement of car meetup events to the browser — featuring event listings, photo galleries, embedded videos, and a community membership form, all wrapped in a clean, responsive layout.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🏠 Hero Section | Eye-catching homepage with event highlights |
| 📅 Upcoming Events | Browse scheduled car meetup events |
| 🖼️ Image Gallery | Curated photos from past meetups |
| 🎥 Video Showcase | Embedded event videos (6 clips) |
| 📝 Membership Form | UI for joining the CarMeet community |
| 📱 Responsive Design | Flexbox & Grid layout adapts to all screen sizes |

## 🚀 Getting Started

No dependencies or build tools needed — this is a pure HTML/CSS project.

**Clone and open:**

```bash
git clone https://github.com/sadiyamulani03/CarMeet-website.git
cd CarMeet-website
```

Then simply open `carmeetup.html` in your browser:

```bash
# On macOS
open carmeetup.html

# On Linux
xdg-open carmeetup.html

# On Windows
start carmeetup.html
```
